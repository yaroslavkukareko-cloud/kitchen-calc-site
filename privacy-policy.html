Privacy Policy

This Privacy Policy explains how the Kitchen Calc browser extension handles data. Kitchen Calc is designed as a simple local tool for cooking measure conversions and proportion calculations.

Plain-language summary

  • Kitchen Calc does not collect personal information.
  • Kitchen Calc does not sell, rent, or share user data.
  • Kitchen Calc does not use analytics, advertising trackers, or third-party tracking services.
  • Kitchen Calc stores only temporary calculation state locally in the browser session.

1. Extension purpose

Kitchen Calc provides a cooking measurement converter and a proportion calculator. It helps users convert kitchen units and calculate recipe-style proportions inside the browser popup.

2. Data handled by the extension

Kitchen Calc does not collect personal data from users. The extension does not require an account, does not ask for a name, email address, location, payment data, or any other personal identifier.

The extension may temporarily store the current calculator state in the browser session so that the user does not lose entered values when closing and reopening the popup.

Data type Example Purpose Storage
Selected ingredient Flour, sugar, water Restore the last converter selection during the browser session Local browser session storage
Selected units US cups, grams, tablespoons Restore the last unit selection during the browser session Local browser session storage
Entered calculator values Amount field and proportion fields Restore the last calculation during the browser session Local browser session storage
Selected tab Measures or Proportions Reopen the popup on the last used tab during the browser session Local browser session storage

3. How data is stored

Kitchen Calc uses Chrome extension storage only for temporary session state. This data is stored locally in the user's browser environment and is used only to restore the calculator state during the same browser session.

The stored values are not transmitted to the developer, to a server, or to third parties.

4. Data sharing and third parties

Kitchen Calc does not share user data with third parties.

  • No analytics providers are used.
  • No advertising networks are used.
  • No tracking pixels are used.
  • No remote database is used.
  • No user data is sold or rented.

5. Network access

Kitchen Calc is designed to work locally in the browser popup. The extension does not require a user account and does not need to send calculation data over the internet.

6. Permissions

Kitchen Calc requests the storage permission only to remember temporary calculator state during the browser session. This permission is not used to collect personal information.

7. Website privacy

This website is a static informational page for Kitchen Calc. It does not include analytics scripts, advertising scripts, tracking cookies, or sign-in functionality.

8. Children's privacy

Kitchen Calc is a general-purpose cooking utility. It is not directed at children and does not knowingly collect personal information from children.

9. Changes to this policy

This Privacy Policy may be updated if Kitchen Calc's functionality changes. If future versions introduce new data handling practices, this policy will be updated to describe those changes.
